## Grade Scale
| Score | Grade | Points | Descriptor |
|-------|-------|--------|------------|
| 97-100 | A+ | 4.0 | Exceptional |
| 90-<97 | A | 4.0 | Excellent |
| 85-<90 | A- | 3.7 | |
| 80-<85 | B+ | 3.3 | |
| 75-<80 | B | 3.0 | Good |
| 70-<75 | B- | 2.7 | |
| 65-<70 | C+ | 2.3 | |
| 60-<65 | C | 2.0 | Fair |
| 57-<60 | C- | 1.7 | |
| 55-<57 | D+ | 1.3 | |
| 52-<55 | D | 1.0 | Poor |
| 50-<52 | D- | 0.7 | |
| <50 | F | 0.0 | Failure |

## Notes
- A+ and A both carry 4.0 grade points.
- For P, I, W - see GRD-003
- This scale applies to students from Fall 2020 onward.
- Postgraduate grading scale differs — see GRD-002.
