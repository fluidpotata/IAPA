# Following grades are without Numerical Value

P: Pass
I: Incomplete
W: Withdrawal

## Notes
- A+ and A both carry 4.0 grade points.
- For P, I, W - see GRD-003
- This scale applies to students from Fall 2020 onward.
- Postgraduate grading scale differs — see GRD-002.
